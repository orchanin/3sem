Файл dishes.txt содержит информацию о видах посуды(1-7) и о их количестве:

<№посуды>:<количество>

0:0 - больше посуды нет.

==========================================================================
Файл washer.txt содержит информацию о том с какой скоростью моет washer 
определенный вид посуды:

<№посуды>:<время, затрачиваемое на мойку>

==========================================================================

Аналогичное содержание имеет файл cleaner.txt

==========================================================================

table.txt - стол, общий файл для общения washer и cleaner. Может вмещать 
ограниченное колличество посуды, максимальный размер которой задается 
как параметр среды TABLE_LIMIT перед стартом

export TABLE_LIMIT=5
==========================================================================
